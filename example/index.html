开源的 San 来自百度 EFE。在 EFE，我们还有其他开源产品：

ECharts
直观，生动，可交互，可高度个性化定制的数据可视化图表。
ESL
比 Require.js 更精简、更高效、更健壮的 AMD Loader。
EDP
提供了前端开发时常用工具：项目管理、包管理、调试、构建、代码生成、代码检测等，并允许用户自定义扩展。
EST
基于 Less 的样式工具库，提供了一系列方便快捷的 mixin，只在调用时才输出代码。
Saber
为移动端与现代浏览器设计的，细粒度、模块化的前端应用框架。
Rebas
与 Saber 搭配使用、基于 Node 的 JavaScript 同构解决方案。
Rider
基于 Stylus 与后处理器，无侵入风格的 CSS 样式工具库。
ER
富浏览器端 Web 应用的框架，适用于并能很方便地构建一个整站式的 AJAX Web 应用。
ESUI
一套功能完善，并具有强大扩展机制的 UI 组件库。基于模板、声明式地编程方式让视图更具表达能力。
ETpl
强复用、灵活、高性能的 JavaScript 模板引擎。
iSlider
表现出众，无任何插件依赖的手机平台 JavaScript 滑动组件。
图说
一个向广大用户提供数据可视化服务平台。
Fontmin
第一个纯 JavaScript 字体子集化方案，让 @font-face 在中文世界成为可能。
FECS
前端代码风格检查套件，包含 HTML、CSS、JavaScript 与 Less 代码的检查与修复。
......
更多来自百度 EFE 的项目与文章。
